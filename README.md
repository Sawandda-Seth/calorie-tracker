# Calorie Counter

## Project Overview

The Calorie Counter is a web application built using HTML, Tailwind CSS, and JavaScript. It allows users to track daily calorie intake by adding, viewing, editing, and removing food items.

## Features

* Add food items and calorie values
* Remove food items
* Edit food items
* Display total calories consumed
* Reset daily calorie records
* Store data using LocalStorage
* Fetch food calorie data from a JSON file using Fetch API
* Responsive user interface built with Tailwind CSS

## Technologies Used

* HTML5
* Tailwind CSS
* JavaScript (ES6)
* Fetch API
* LocalStorage
